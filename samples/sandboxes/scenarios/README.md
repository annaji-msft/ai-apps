# Sandbox scenarios

Composed use cases that combine multiple sandbox capabilities. Each
scenario is a placeholder right now — start with the [guides](../guides/)
for runnable code today.

| # | Scenario | What it will show |
|---|---|---|
| 01 | [webapps](01-webapps) | Web apps in sandboxes — server + public port + client |
| 02 | [coding-agents](02-coding-agents) | Run a coding agent (Claude Code / Codex / Copilot CLI) per task in a fresh sandbox |
| 03 | [code-interpreter](03-code-interpreter) | LLM-driven code execution — generate, run, observe, iterate |
| 04 | [swarms](04-swarms) | Many sandboxes, one orchestrator — fan-out work across N workers |
| 05 | [data-processing](05-data-processing) | Producer/consumer pipelines on shared AzureBlob volumes |
| 06 | [developer-workflows](06-developer-workflows) | PR builds, ephemeral CI, on-demand dev environments |
| 07 | [computer-use](07-computer-use) | Browser/desktop automation inside a sandbox for agentic UI tasks |
